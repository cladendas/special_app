# company-project

web-сервер, который принимает разные запросы, парсит тело

На текущий момент работает с эндпоинтами:
/raw POST - сохраняет информацию о запросе в файл
/raw GET - возращает содержимое файла после, оставшее после запроса на /raw POST
/marquiz POST - принимает запрос от данного сервиса, парсит
